Центр инструкций

Ollama и CORS

Чтобы браузер мог обращаться к локальному Ollama, задайте переменную окружения перед запуском:

OLLAMA_ORIGINS="*"

В macOS/Linux: export OLLAMA_ORIGINS="*" затем перезапустите Ollama. Приложения на HTTPS (Puter) к HTTP localhost могут быть ограничены политикой браузера — при необходимости используйте туннель или Puter ИИ.

Ключи API: Gemini и OpenAI

  • OpenAI: console.openai.com → API keys → создать ключ, вставьте в панели в режиме «OpenAI».
  • Gemini: aistudio.google.com/apikey → создать ключ Google AI Studio, режим «Google Gemini».
  • Anthropic: console.anthropic.com → API Keys.

Vibe Coding в Puter

В среде Puter откройте встроенный редактор/терминал, подключите ИИ через эту панель или встроенный puter.ai. Описывайте задачу естественным языком, итерируйте с подсказками, храните проекты в облачной ФС и клонируйте репозитории через «Клонировать репозиторий».
